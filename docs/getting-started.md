# Getting Started

Quick start guide for the Fuel Dashboard

## First Steps

### 1. Access the Dashboard
Navigate to your Fuel Dashboard instance. You'll see the main dashboard with:
- KPI overview cards
- Interactive map of sites
- Due deliveries table
- Today's schedule

### 2. Understand the Layout

**Top Section**
- Site totals and quick metrics
- KPI visual overview
- Search and export functions

**Left Column**
- Today's date and time
- Search box for finding sites
- Download data button
- KPI summary cards

**Right Column**
- Interactive map with site locations
- Map controls (zoom, layers)
- Site status indicators

**Bottom Section**
- Due/Overdue deliveries
- Today's scheduled deliveries
- Detailed site information

## Navigation

### Dashboard Sections

#### Today
View all fueling deliveries scheduled for **today**
- Prioritize immediate operations
- See preparation status
- Track completion progress

#### Tomorrow
View deliveries for **tomorrow**
- Plan ahead
- Check resource availability
- Confirm schedules

#### Coming in 3 Days
View upcoming deliveries in the **next 3 days**
- Medium-term planning
- Resource allocation
- Schedule confirmation

#### Due/Behind
View **overdue or behind schedule** deliveries
- Priority management
- Reschedule operations
- Document delays

## Using the Map

### Map Controls
- **Zoom In/Out**: Use + and - buttons
- **Pan**: Click and drag the map
- **Layers**: Toggle different map styles
- **Overlays**: Show/hide sites layer

### Map Features
- **Click a Site**: View site details in popup
- **Color Coding**: Understand site status by color
- **Legend**: See what each color means

### Map Layers
- **Street Map**: Default view
- **Satellite**: Aerial imagery
- **Dark Mode**: Night mode
- **Terrain**: Topographic view

## Searching & Filtering

### Quick Search
1. Click the search box
2. Type site name or ID
3. Results filter in real-time
4. Click a result to view details

### Filter by Status
- **Today**: Active deliveries now
- **Due**: Overdue items only
- **Upcoming**: Future deliveries
- **All**: Show everything

## Viewing Site Details

### From Table
1. Click on any site row
2. View details popup (if enabled)
3. Check scheduled date
4. Review site location

### From Map
1. Click a site marker
2. Popup shows:
   - Site name
   - Current status
   - Delivery date
   - Site ID

## Exporting Data

### Download Fueling Data
1. Click **Download** button
2. Select data format:
   - CSV (spreadsheet)
   - PDF (printable)
3. Choose date range or current view
4. File downloads to your computer

### Using Exported Data
- Spreadsheet analysis
- Report generation
- Data backup
- Third-party integration

## Working with KPI Metrics

### Understanding the Numbers
- **Total Sites**: All registered locations
- **Today**: Count of deliveries today
- **Coming in 3 Days**: Upcoming deliveries
- **Due (Overdue)**: Behind schedule count

### KPI Chart
- Blue: On-time deliveries
- Red: Due/overdue deliveries
- Shows ratio and trends

## Auto-Refresh

The dashboard automatically updates every **2 minutes**. This means:
- New sites appear automatically
- Status changes update in real-time
- No need to manually refresh
- Always see current information

## Common Tasks

### Finding a Specific Site
1. Use search box
2. Type site name or ID
3. View in list or on map
4. Click to see details

### Checking Today's Deliveries
1. Click "Today" tab
2. View all today's sites
3. Check status and times
4. Download if needed

### Viewing Overdue Items
1. Click "Due/Behind" tab
2. See all overdue items
3. Prioritize accordingly
4. Plan rescheduling

### Locating Sites Geographically
1. View interactive map
2. Click site markers
3. See location details
4. Zoom in/out as needed

### Exporting Site Data
1. Click Download button
2. Select format and range
3. Confirm selection
4. Save to computer

## Troubleshooting

### Map Not Showing
- Check internet connection
- Try refreshing page
- Clear browser cache
- Try different browser

### Data Not Updating
- Wait for 2-minute auto-refresh
- Manually refresh page
- Check data source
- Contact support

### Search Not Working
- Try partial match
- Check spelling
- Clear search box
- Refresh page

### Export Issues
- Ensure pop-ups allowed
- Check browser settings
- Try different format
- Clear temporary files

## Keyboard Shortcuts

- `Ctrl+F`: Focus search
- `Escape`: Clear search
- `Enter`: Submit search
- `Tab`: Navigate elements

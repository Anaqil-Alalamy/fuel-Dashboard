# Fuel Dashboard

A comprehensive fueling plan management system for GSM sites.

## Features

- **Dashboard**: Track fueling schedules across all sites
- **Multi-site Management**: Manage fuel deliveries for multiple locations
- **Status Tracking**: Monitor fueling status in real-time
- **Flexible Scheduling**: Plan fuel deliveries by date categories (Today, Tomorrow, Coming in 3 Days, Due/Behind)

## Getting Started

Visit the [Fuel Dashboard]({{ site.baseurl }}/) to manage your fueling operations.

## Categories

- **Today**: Fuel deliveries scheduled for today
- **Tomorrow**: Fuel deliveries scheduled for tomorrow
- **Coming in 3 Days**: Upcoming deliveries in the next 3 days
- **Due / Behind**: Overdue or behind schedule deliveries

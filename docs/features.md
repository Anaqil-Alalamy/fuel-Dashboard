# Features

Comprehensive fueling plan management for GSM Sites

## Dashboard Overview

### Real-time KPI Metrics
- **Total Sites**: Count of all registered fueling sites
- **Today**: Active deliveries scheduled for today
- **Coming in 3 Days**: Upcoming deliveries within next 3 days
- **Due/Overdue**: Behind schedule or overdue deliveries

### KPI Chart
Visual donut chart showing:
- Due sites percentage
- On-time sites percentage
- Historical comparison

## Site Management

### Multi-Site Tracking
- Manage unlimited fueling sites

- Location-based organization
- Bulk operations support

### Site Information
- Site identification (Name/ID)

## Scheduling System

### Scheduling Categories
1. **Today** - Immediate deliveries
2. **Tomorrow** - Next-day deliveries
3. **Coming in 3 Days** - Medium-term scheduling
